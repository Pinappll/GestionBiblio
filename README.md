# GestionBiblio

Gestion des livres d'une bibliothèque.

## Prérequis

- PHP 7.4 ou supérieur
- Composer

## Installation

1. Clonez le dépôt :

   ```sh
   https://github.com/Pinappll/GestionBiblio.git
   cd GestionBiblio
   ```

2. Installez les dépendances avec Composer :

   ```sh
   composer install
   ```

## Utilisation

Pour exécuter le projet, lancez le fichier index.php :

```sh
php index.php
```
